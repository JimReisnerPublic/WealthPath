## Stack at a Glance

```
Request
  │
  ▼
FastAPI router (api/routers/)
  │  Pydantic validates request body
  │
  ▼
AIEngine (services/ai_engine.py)
  │
  ├─ /explain  ──► LCEL chain: ChatPromptTemplate | AzureChatOpenAI | StrOutputParser
  │                (LangChain — equivalent of SK kernel.invoke_prompt)
  │
  └─ /plan     ──► LangGraph ReAct agent
                   (LangGraph — equivalent of AutoGen AssistantAgent)
                     │
                     ├─ get_cohort_median_income    (@tool wrapping SCFDataService)
                     ├─ get_cohort_median_net_worth
                     ├─ get_median_projection       (@tool wrapping SimulationEngine)
                     └─ evaluate_retirement_plan    (@tool wrapping SurrogateModelService)
```

## Useful Commands

```bash
# Run the API server with live reload
uvicorn wealthpath.main:app --reload

# Interactive API docs (auto-generated by FastAPI)
open http://localhost:8000/docs

# Run tests
pytest -v

# Run a single test file
pytest tests/test_simulation_engine.py -v

# Generate ML training data (SCF-anchored — requires real SCF parquet)
python scripts/generate_training_data.py --scf data/scf_2022.parquet

# Generate ML training data (pure synthetic fallback — no SCF data required)
python scripts/generate_training_data.py --n 50000

# Train the surrogate model (run after generate_training_data.py)
python scripts/train_surrogate_model.py

# View MLflow experiment results
python -m mlflow ui   # → http://localhost:5000
```
